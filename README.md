# DeXiv

**DeXiv** is a Decentralized Scientific Research Platform designed to revolutionize the way research is conducted and shared. By leveraging decentralized technologies, DeXiv provides a secure, transparent, and collaborative environment for scientific research.

## Features

- **Decentralization:** Eliminates single points of failure by distributing data across the network.
- **Collaboration:** Enables seamless global collaboration among researchers.
- **Transparency:** Ensures data integrity and reproducibility in scientific research.
- **Scalability:** Built to accommodate growing datasets and increasing user demands.

## Prerequisites

Before getting started, ensure you have the following installed:

- **Node.js:** Version 12 or higher. You can download it from [nodejs.org](https://nodejs.org/).

## Installation

Clone the repository and install the dependencies:

```bash
git clone https://github.com/your-username/DeXiv.git
cd DeXiv
npm install
```

## Running the Application

After installing the dependencies, launch the development environment with:

```bash
npm run dev:all
```

Once the server is up and running, open your browser and navigate to:

[http://localhost:3001]

You should now see the DeXiv platform in action.
